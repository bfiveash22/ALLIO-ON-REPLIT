## 📡 API ENDPOINT SPECIFICATION

### Option 1: Database-Based Communication (RECOMMENDED)

**How It Currently Works:**
- Allio agents write to `openclaw_messages` database table
- OPENCLAW polls table every 1-5 minutes (based on priority)
- OPENCLAW sends response back to Trustee via WhatsApp/Telegram

**Database Table Schema:**
```sql
CREATE TABLE openclaw_messages (
  id SERIAL PRIMARY KEY,
  agent_id VARCHAR(50) NOT NULL,
  message TEXT NOT NULL,
  priority VARCHAR(10) DEFAULT 'normal',
  status VARCHAR(20) DEFAULT 'pending',
  created_at TIMESTAMP DEFAULT NOW(),
  sent_at TIMESTAMP
);
```

**To Send a Task to OPENCLAW:**
```javascript
// From any Allio agent
await db.insert('openclaw_messages', {
  agent_id: 'AGENT_4',
  message: 'Task description here',
  priority: 'high' // options: urgent, high, normal, low
});
```

**OPENCLAW polls and relays to Trustee automatically.**

## 🛠️ IMPLEMENTATION STEPS FOR AGENT 4

### Step 1: Create API Route in Allio

**File:** `server/routes/openclaw.ts` (or add to existing routes)

```typescript
import { Router } from 'express';
import { db } from '../db';

const router = Router();

// Endpoint to send tasks to OPENCLAW
router.post('/api/openclaw/send', async (req, res) => {
  const { agent_id, task, priority, context, callback_url } = req.body;

  try {
    // Insert into database for OPENCLAW polling
    const message = await db.insert('openclaw_messages', {
      agent_id,
      message: JSON.stringify({ task, context }),
      priority: priority || 'normal',
      callback_url,
      status: 'pending'
    });

    res.json({
      success: true,
      message_id: message.id,
      status: 'sent_to_trustee',
      estimated_response_time_minutes: 30
    });
  } catch (error) {
    res.status(500).json({ success: false, error: error.message });
  }
});

// Endpoint to receive OPENCLAW responses
router.post('/api/openclaw/response', async (req, res) => {
  const { message_id, status, result } = req.body;

  try {
    // Update message status
    await db.update('openclaw_messages',
      { id: message_id },
      { status, result: JSON.stringify(result), sent_at: new Date() }
    );

    // Trigger UI update or notification
    // emit('openclaw_response', { message_id, result });

    res.json({ success: true });
  } catch (error) {
    res.status(500).json({ success: false, error: error.message });
  }
});

export default router;
```

### Step 3: Add Helper Function for Agents

**File:** `server/services/openclaw.ts`

```typescript
export async function sendToOpenClaw(
  agentId: string,
  task: string,
  options?: {
    priority?: 'urgent' | 'high' | 'normal' | 'low';
    context?: any;
    callbackUrl?: string;
  }
) {
  const response = await fetch('https://ffpma.com/api/openclaw/send', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      agent_id: agentId,
      task,
      priority: options?.priority || 'normal',
      context: options?.context,
      callback_url: options?.callbackUrl || 'https://ffpma.com/api/openclaw/response'
    })
  });

  return response.json();
}
```

**Usage Example:**
```typescript
// From any Allio agent
import { sendToOpenClaw } from '../services/openclaw';

async function requestProtocol(patientName: string) {
  const result = await sendToOpenClaw(
    'DR_FORMULA',
    `Create protocol for ${patientName}`,
    {
      priority: 'high',
      context: { patient_name: patientName, diagnosis: 'Cancer' }
    }
  );

  console.log('Sent to OPENCLAW:', result.message_id);
}
```

## 🚨 ERROR HANDLING

### Possible Error States

1. **`status: 'pending'`** - Waiting for OPENCLAW to pick up
2. **`status: 'processing'`** - OPENCLAW is working on it
3. **`status: 'completed'`** - Task finished successfully
4. **`status: 'failed'`** - Task failed (check `result` field for error details)
5. **`status: 'timeout'`** - No response after 4 hours

**Handle Timeouts:**
```javascript
// After 4 hours with no response, mark as timeout
const TIMEOUT_HOURS = 4;

setInterval(async () => {
  const cutoff = new Date(Date.now() - TIMEOUT_HOURS * 60 * 60 * 1000);

  await db.update('openclaw_messages',
    { status: 'pending', created_at: { $lt: cutoff } },
    { status: 'timeout' }
  );
}, 60 * 60 * 1000); // Check every hour
```
