# Chatbot Authentication - Members Only Access

**✅ AUTHENTICATION ENABLED**

The chatbot now requires users to be logged into WordPress before they can use it.

---

## How It Works

1. **Widget checks for WordPress login cookie**
   - Looks for `wordpress_logged_in_*` cookie
   - If not found, chat widget is hidden

2. **API requires authentication**
   - Every chat request must include WordPress session
   - Unauthenticated requests get 401 error

3. **Members-only feature**
   - Only logged-in FFPMA members can use chatbot
   - Non-members see nothing (widget hidden)

---

## WordPress Integration

The widget automatically:
- Detects if user is logged in
- Sends session cookie to API
- Shows login prompt if session expires

**No additional configuration needed** - WordPress session detection is automatic.

  -H "Content-Type: application/json" \
  -d '{"message":"test"}'

**Test with API key (for development):**
```bash
API_KEY=$(grep CHATBOT_API_KEY .env.chatbot | tail -1 | cut -d= -f2)
curl -X POST http://130.49.160.73:3500/api/chat \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $API_KEY" \
  -d '{"message":"test"}'
```

## Behavior

**Logged Out Users:**
- Widget is hidden (display: none)
- No chat button visible
- Console message: "Please log in to use the product expert"

**Logged In Members:**
- Widget visible bottom-right
- Chat button clickable
- Full chatbot functionality

**Session Expires During Chat:**
- Next message shows login prompt
- Prompts user to re-login
- No data lost (conversation stored client-side)

---

## Security

**What's Protected:**
- All chat endpoints require auth
- Product search requires auth
- No anonymous access

**What's Public:**
- `/health` endpoint (for monitoring)
- Static widget files (but won't work without auth)

**WordPress Session Verification:**
- Checks for valid `wordpress_logged_in_` cookie
- Verifies with WordPress REST API
- Only allows active member sessions

## Deployment Notes

**WordPress Embed Code (Updated):**
```html
<script>
  window.FFPMA_CHAT_API = 'http://130.49.160.73:3500';
  
  (function() {
    const script = document.createElement('script');
    script.src = window.FFPMA_CHAT_API + '/widget.html';
    script.async = true;

  })();
</script>
```

**No changes needed to embed code** - authentication is handled automatically.
